API v1.0.1:
-----------
La siguiente actualización incorpora los métodos GET /user/current/routines/ y /user/{userId}/routines/.

Instalación de la actualización del API:
----------------------------------------
1) Descomprimir el archivo "API v1.0.1 Installer.zip" en el directorio de instalación.
2) Reiniciar el API para que se aplique la actualización.
NOTA: Para corroborar la correcta instalación de la actualización verificar que en el pie de página de la documentación del API se muestre la leyenda "Swagger Exercise 1.0.1".
